# EKIA
A Furniture E-Commerce app built with Angular.

You can visit the live Demo on https://ekia-9249e.web.app/.

## Built with
```Angular```-```Tailwind```-```Firebase```.
### Other Libraries
```Font Awesome```

## Features
- Home page shows a random product as a main hero.

## Usage
### Installing dependencies
```
npm i
```
### Run
```
ng serve
```
